# Proyecto CanSat 2026

## Descripción del Proyecto
Este proyecto tiene como objetivo el diseño, construcción y vuelo de un CanSat para la competencia 2026. La misión principal consiste en un sistema de transporte y entrega de un instrumento delicado (un huevo de gallina) mediante un paracaídas y un paraglider controlado.

## Estructura del Repositorio
- `contracts/`: Contratos y términos de convivencia del equipo.
- `investigation/`: Documentación, investigaciones técnicas y plantillas.
- `investigation/templates/`: Plantillas para nuevas investigaciones.

## Reglas Principales
- **Compromiso:** 6 horas semanales mínimo.
- **Reuniones:** Reunión semanal obligatoria.
- **Comunicación:** Responder mensajes en menos de 24 horas.
- **Propiedad Intelectual:** Todo el trabajo pertenece al equipo.
